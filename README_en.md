# Dreamina CLI (dreamina-cli) Skill

## Project Introduction

Dreamina CLI Skill is a professional assistant to help users install, login, and use the Dreamina CLI for various AI generation tasks. It provides multiple functional modules including text-to-image, text-to-video, image-to-image, image-to-video, and multimodal reference (multimodal2video) modes.

## Features

### 1. Installation and Login Management
- Check if Dreamina CLI is installed
- Guide users through login process
- Query account balance
- Manage login status

### 2. Text-to-Image/Text-to-Video
- Generate images from text descriptions
- Generate videos from text descriptions
- Support multiple resolutions and aspect ratios
- Provide common prompt patterns

### 3. Image-to-Image/Image-to-Video
- Generate style-converted images from existing images
- Generate dynamic videos from single images
- Support image path checking and validation

### 4. Multimodal Reference (multimodal2video)
- Generate videos from images, reference videos, and background music
- Support Seedance 2.0 model
- Provide complex multimodal generation capabilities

### 5. Task Management
- View task lists
- Query task results
- Download generated content
- Support task status filtering

## Usage

### Basic Usage

1. **Activate the Skill**: Input "即梦auto启动" (Chinese trigger phrase)
2. **Select Function Module**:
   - 1. Installation and Login
   - 2. Text-to-Image/Text-to-Video
   - 3. Image-to-Image/Image-to-Video
   - 4. Multimodal Reference
   - 5. Task Management
3. **Provide Detailed Requirements**: Based on the selected module, provide specific requirements

### Example Usage

```
User: 即梦auto启动，我要生成一张"一只戴墨镜的橘猫"的图片，1:1比例，2k分辨率

Skill: OK, text-to-image function is activated. I will help you generate this image.

Executes command:
dreamina text2image --prompt="一只戴墨镜的橘猫" --ratio=1:1 --resolution_type=2k --poll=30
```

## Best Practices

1. **Use absolute paths**: All material files should use absolute paths
2. **Prefer --poll parameter**: Let tasks automatically wait for results
3. **Save submit_id**: For subsequent querying and retries
4. **Set reasonable polling times**:
   - Image tasks: 30-60 seconds
   - Video tasks: 180-300 seconds
   - Multimodal reference: 180-300 seconds

## Technical Support

### Common Issues

- **Login problems**: Check ~/.dreamina_cli/config.toml file
- **Task failures**: Verify material paths and formats
- **Task timeouts**: Increase --poll parameter duration
- **Skill not triggering**: Ensure using correct trigger phrase "即梦auto启动"
