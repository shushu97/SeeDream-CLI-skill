# 即梦 CLI (dreamina-cli) 技能

## 项目简介

即梦 CLI 技能是一个帮助用户安装、登录和使用即梦 CLI 进行各种 AI 生成任务的专业助手。它提供了多种功能模块，包括文生图、文生视频、图生图、图生视频和全能参考（multimodal2video）模式。

## 功能特性

### 1. 安装与登录管理
- 检查即梦 CLI 是否安装
- 指导用户完成登录流程
- 查询账户余额
- 管理登录状态

### 2. 文生图/文生视频
- 从文字描述生成图片
- 从文字描述生成视频
- 支持多种分辨率和比例
- 提供常用的提示词模式

### 3. 图生图/图生视频
- 从图片生成风格转换的图片
- 从单张图片生成动态视频
- 支持图片路径检查和验证

### 4. 全能参考 (multimodal2video)
- 从图片、参考视频和背景音乐生成视频
- 支持 Seedance 2.0 模型
- 提供复杂的多模态生成功能

### 5. 任务管理
- 查看任务列表
- 查询任务结果
- 下载生成的内容
- 支持任务状态筛选

## 使用方法

### 基本使用

1. **激活技能**：输入"即梦auto启动"
2. **选择功能模块**：
   - 1. 安装与登录
   - 2. 文生图/文生视频
   - 3. 图生图/图生视频
   - 4. 全能参考
   - 5. 任务管理
3. **提供详细需求**：根据选择的功能模块，提供具体的需求信息

### 示例使用

```
用户：即梦auto启动，我要生成一张"一只戴墨镜的橘猫"的图片，1:1比例，2k分辨率

技能：好的，文生图功能已启动。我来帮您生成这张图片。

执行命令：
dreamina text2image --prompt="一只戴墨镜的橘猫" --ratio=1:1 --resolution_type=2k --poll=30
```
